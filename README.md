## Installation

### Homebrew (macOS and Linux)

```bash
brew install claylo/brew/test-crate
```

### Pre-built Binaries

Download the latest release for your platform from the [releases page](https://github.com/claylo/test-crate/releases).

### From Source

```bash
cargo install test-crate
```

Or build from source:

```bash
git clone https://github.com/claylo/test-crate.git
cd test-crate
cargo install --path crates/test-crate
```
